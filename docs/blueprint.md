# **App Name**: Git Insights

## Core Features:

- Repo URL Input: Allows the user to input a GitHub repository URL.
- Fork Count Display: Displays the total number of forks for the given repository.
- Commit Count per Fork: Displays the number of commits for each fork of the repository.
- Repo Link: Provides a direct link to each forked GitHub repository.
- Recent Commits: Shows the most frequent committers over the last 48 hours.
- Github Pages support: Can be deployed to Github Pages directly.

## Style Guidelines:

- Primary color: `#5162FB`, a saturated blue to suggest intelligence and github's brand.
- Background color: `#E7E9FD`, a light tint of blue for a clean feel.
- Accent color: `#A259FF`, a vivid violet to guide the user.
- Font pairing: 'Space Grotesk' for headlines and 'Inter' for body text. Space Grotesk is a sans-serif. Inter is a sans-serif.
- Simple, clear icons from a set like Font Awesome or Material Icons to represent forks, commits, and filtering options.
- A clean, tabular layout to display repository information, with clear sections for forks, commit counts, and links.
- Subtle transitions when loading data or filtering results to provide a smooth user experience.
